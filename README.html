Materials for the workshop R & The Graphical Web at The Graphical Web 2015 conference.

On Saturday, September 26th, I had the privilege to conduct a 3 hour workship on R & The Graphcial Web conference. This repo contains all the materials used to make the presentation.

The ioslides presentation has presenter notes peppered throughout various pages. Dual-screen view it or hit p on a slide to see if there are any.

The widgety directory contains a simple expanded htmlwidget that builds a customizable D3 circle that was used in the second half of the workshop.
